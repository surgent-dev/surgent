# Security

Please report suspected vulnerabilities privately to the maintainers before
posting public details.

Use GitHub private vulnerability reporting for this repository. If that is not
available, contact the maintainers privately before sharing details publicly.

Do not include production secrets, customer data, API tokens, webhook payloads,
or database exports in issues or pull requests.

For local development, keep secrets in ignored `.env.local` files. The tracked
`.env.example` files are placeholders only.
